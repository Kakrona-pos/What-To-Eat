# 🍳 What To Eat? || Recipe Recommendation System

An AI-powered cooking assistant built with **Streamlit** and **Gemini 2.5 Flash**. Input your ingredients, set your cooking profile, and get tailored recipes instantly.

## 🚀 How to Run Locally
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## 🔑 Setup
You will need a **Google Gemini API Key**. You can get one from the [Google AI Studio](https://aistudio.google.com/).
